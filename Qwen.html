인공지능의 역사

하드웨어 발전과 네트워크 기술의 진화 속에서 본 AI의 여정

1940년대: 기반 마련
진공관 컴퓨터

1946년 최초의 전자 컴퓨터 에니악(ENIAC) 등장. 17,468개 진공관 사용

진공관은 스위치 역할을 하며 0과 1을 표현. 하지만 크기와 전력 소모가 컸음

트랜지스터 발명

1947년 벨연구소에서 트랜지스터 개발. 진공관의 단점을 극복

주요 인물:
  • 존 애타나소프 - 최초의 전자 컴퓨터 설계자
  • 존 폰 노이만 - 프로그램 내장식 컴퓨터 개념 제안
1950년대: AI 탄생
다트머스 회의

1956년 최초의 AI 학술 회의. "인공지능"이라는 용어 공식 사용 시작

퍼셉트론

1957년 프랑크 로젠블랏의 신경망 모델. IBM 704에서 구현됨

주요 인물:
  • 존 매카시 - "인공지능" 용어 창시자
  • 마빈 민스키 - 기호주의 대표 학자
1960년대: AI의 겨울
XOR 문제

1969년 민스키에 의해 단층 퍼셉트론의 한계 증명. 연구 자금 감소

실리콘밸리 탄생

윌리엄 쇼클리의 반도체 연구소에서 인텔 창립으로 이어짐

기술 발전:
  • 1958년: 잭 킬비의 IC(집적회로) 발명
  • 1969년: DARPA의 ARPANET 시작
1970-80년대: 부활의 시작
역전파 알고리즘

1986년 제프리 힌튼 팀이 다층 신경망 학습 방법 증명

집적회로 발전

인텔의 마이크로프로세서(1971)와 D-RAM(1973)로 컴퓨터 성능 혁신

주요 인물:
  • 제프리 힌튼 - 딥러닝의 아버지
  • 얀 루쿤 - CNN 개발
1990년대: 인터넷 시대
인터넷 확산

1991년 팀 버너스리의 월드와이드웹으로 데이터 공유 혁명

GPU 등장

1999년 엔비디아의 GPU로 병렬 처리 능력 비약적 발전

주요 조직:
  • IBM - 1401 컴퓨터 보급
  • DARPA - ARPANET 주도
2000년대: 데이터 시대
이미지넷 프로젝트

2009년 시작된 대규모 이미지 데이터베이스로 딥러닝 발전 기여

클라우드 컴퓨팅

2006년 AWS 시작으로 분산 처리 환경 조성

기술 발전:
  • 2007년: iPhone 출시로 모바일 데이터 급증
  • 2011년: IBM 왓슨의 제퍼디 우승
2010년대 이후: 딥러닝 시대
알파고

2016년 이세돌과의 바둑 대결로 AI 가능성 증명

트랜스포머

2017년 구글의 트랜스포머 모델로 NLP 혁명

최근 발전:
  • 2020년: GPT-3의 언어 생성 능력
  • 2022년: 챗GPT의 대중화
